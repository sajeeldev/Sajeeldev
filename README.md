## 🚀 Featured Project

### ⚖️ [OneOps.ai — Legal Practice Management Platform](https://github.com/sajeeldev)
> An enterprise-grade SaaS platform built on **Laravel 10** that gives law firms a complete digital operations suite — from matter management and billing to HR, workflows, and client portals.

- 🔄 **DAG-based visual workflow engine** with 12 node action types (Email, Approval, HTTP Request, Decision, AI Task Generation, and more)
- 🏢 **Multi-tenant SaaS architecture** with company isolation, package-based subscriptions, and module-based licensing
- 🔐 **Granular RBAC** — 4 role tiers with 5 permission levels (All / Added / Owned / Both / None) powered by custom access control
- 💳 **10+ payment gateway integrations** including Stripe, PayPal, Razorpay, and Mollie with recurring billing support
- 🌍 **20+ language support** with RTL, multi-currency, Google Calendar/Outlook sync, and Zoom integration
